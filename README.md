# 🚀 AI YouTube Shorts Automation Bot

Automated system that transforms music into viral-ready YouTube Shorts using AI.

---

## 🎯 Overview

This project is an end-to-end automation pipeline that:

- Processes music files from Google Drive
- Generates dynamic short-form videos with beat-synced editing
- Uses AI to create high-quality visuals (anime-style aesthetic)
- Automatically generates titles, descriptions, and tags
- Uploads videos directly to YouTube
- Manages content queue and posting logic

---

## ⚙️ Features

- 🎧 Automatic music style detection
- 🎬 Dynamic video editing (zoom, shake, beat sync, drop effects)
- 🧠 AI-generated visuals (Claude + Flux)
- 👩 High-retention visual strategy (centered character focus)
- 📤 Automatic YouTube uploads
- 🔄 Smart queue system (multiple shorts per track)
- ☁️ Google Drive integration
- 🤖 AI-generated metadata (titles, descriptions, tags)

---

## 🧠 Technologies Used

- Python
- FFmpeg
- GitHub Actions (automation)
- Google Drive API
- YouTube Data API
- Replicate (image generation)
- Claude (AI text & prompt generation)

---

## 🔁 Workflow

1. Fetch music from Google Drive
2. Detect style and generate visual prompt
3. Generate AI image
4. Create short video with dynamic effects
5. Generate metadata using AI
6. Upload to YouTube automatically

Designed to scale content production for short-form platforms like:
